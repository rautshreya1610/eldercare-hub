🧓 ElderCare Hub – Smart Elder Assistance Platform
📌 Overview

ElderCare Hub is a web-based assistance platform designed to support elderly individuals in managing their daily activities independently. The system provides a simple and accessible interface that enables senior citizens to book household helpers, receive basic medical guidance through an AI chatbot, and set reminders for important tasks.

The platform focuses on usability, accessibility, and simplicity, ensuring that elderly users with limited technical knowledge can interact with the system comfortably.

By integrating service assistance and health guidance into a single platform, ElderCare Hub helps improve the quality of life and independence of senior citizens.

🎯 Project Objectives

Improve digital accessibility for elderly users with simple UI and large readable text.

Provide basic medical guidance through an AI chatbot in easy language.

Enable seniors to book helpers for daily household tasks such as cooking, cleaning, and laundry.

Create a scalable and modular system architecture that can support future features like emergency alerts and doctor consultations.

🚀 Key Features
1️⃣ Helper Booking System

The platform allows elderly users to find and schedule reliable helpers for household services.

Features:

Browse helpers by service type (cleaning, cooking, laundry)

View helper profiles with details

Book helpers easily

Real-time booking updates

This helps seniors manage daily tasks without depending completely on family members.

2️⃣ AI Medical Chatbot

The system includes an AI-powered chatbot that helps users understand basic health-related information.

Features:

Answers common health-related questions

Uses simple and clear language suitable for seniors

Provides safe medical guidance

Encourages consultation with healthcare professionals when needed

3️⃣ Reminder & Notification System

This module helps elderly users remember important daily tasks.

Features:

Medication reminders

Activity reminders

Scheduled notifications

This improves routine management and reduces the chances of missing important tasks.

4️⃣ Elderly Friendly User Interface

The interface is designed specifically for elderly users.

Features:

Large readable fonts

High contrast design

Simple navigation

Minimal steps to perform actions

These features make the platform easy to use for users with limited technical experience.

🏗️ System Architecture

The system consists of the following layers:

Frontend Layer

Provides the user interface for elderly users.

Backend Layer

Handles application logic and user requests using Flask.

Database Layer

Stores user data, helper details, and booking information using Firebase.

AI Processing Layer

Handles chatbot responses and medical query processing.
